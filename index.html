Went Exchange

Plateforme de snippets de code

Un snippet est un bout de code ayant une fonction précise : vérifier si une mise à jour est disponible, envoyer une notification, obtenir le contenu d'un fichier JSON en ligne, etc.
Exchange répertorie plusieurs snippets de code publiés à partir de produits Went. Cela représente un énorme atout. Prenons l'exemple d'Actuality Feed : pour que l'add-on informe l'utilisateur qu'une mise à jour est disponible, elle possède un fichier update.js.
Comme update.js fonctionne correctement, le copier permet d'avoir des fonctionnalités rapidement et sans bugs !
